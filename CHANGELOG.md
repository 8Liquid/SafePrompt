# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Initial release of SafePrompt API
- 13 moderation categories (profanity, hate speech, toxicity, violence, threats, self-harm, NSFW, scam/phishing, PII, spam, prompt injection, jailbreak, all-caps aggression)
- Text sanitization and cleaning capabilities
- Scoring system with severity levels (low, medium, high)
- OpenAPI specification
- Comprehensive test suite
- Full documentation (README, CONTRIBUTING, CODE_OF_CONDUCT)
